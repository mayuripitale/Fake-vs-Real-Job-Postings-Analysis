# Fake-vs-Real-Job-Postings-Analysis

**Project Overview**
This project analyzes fake and real job postings to identify fraud patterns in online recruitment data. The focus is on understanding how fraudulent postings differ from legitimate ones by examining key attributes and presenting actionable insights through an interactive Power BI dashboard.

**Tools & Technologies**

- Excel – Data cleaning, preprocessing, and validation

- Power BI – Data modeling, analysis, and dashboard development

**Analytical Approach**

- Reviewed and prepared raw job posting data for analysis

- Cleaned missing, inconsistent, and irrelevant records in Excel

- Structured and modeled data for reporting

- Developed interactive Power BI visuals to compare fake and real postings

- Analyzed trends across country, industry, employment type, and work mode

**Key Insights**

- A substantial share of the job postings in the dataset were fraudulent

- Fraudulent postings are concentrated in certain countries

- Remote and contract roles are more likely to be fraudulent
  
- Fake job postings frequently lack clear company and salary information

**Dashboard Overview**

- Fake vs real job posting comparison

- Country-wise fraud distribution

- Employment type and work mode analysis

- Interactive slicers for detailed exploration
  
**Conclusion**

This project demonstrates how data analytics can uncover patterns in fraudulent job postings and presents insights in a clear, interactive dashboard. The findings highlight key trends and enable data-driven understanding of fake vs real job postings.
